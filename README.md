## 🔥 Featured Project

### 🧠 BeyondWords — Real-Time Sign Language Interpretation
A full-stack AI-powered video conferencing platform that translates sign language into text/speech in real time.

**✨ Highlights:**
- 🤖 AI gesture recognition using MediaPipe + PyTorch (LSTM)
- 🎥 Real-time communication using WebRTC & Socket.io
- 🌍 Multi-language translation (English, Hindi, German)
- 📝 Live transcription with speech-to-text
- 🎨 Modern UI with React + Tailwind CSS

**🛠 Tech Stack:**  
`React` • `Node.js` • `Socket.io` • `WebRTC` • `Python` • `PyTorch` • `MediaPipe` • `OpenCV`
